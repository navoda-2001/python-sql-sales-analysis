# python-sql-sales-analysis




This project demonstrates how to combine **SQL and Python** for data analysis.  
A simple sales dataset is stored in a SQLite database, queried using SQL, and analyzed using Python (Pandas).  
The results are visualized using Matplotlib to understand customer spending patterns.

This project simulates a real-world data analyst workflow used in companies.

---

##  Tools & Technologies
- Python
- SQLite
- Pandas
- Matplotlib
- Google Colab
- GitHub

---

## 📂 Project Workflow

###  Create Database
A SQLite database (`sales.db`) was created to store sales data.

###  Create Table
A Sales table was created with:
- Order_ID
- Customer_ID
- Amount

###  Insert Data
Sample sales records were inserted into the database.

###  SQL Query Execution
SQL queries were executed using Python to:
- Calculate total spending per customer
- Group data using `GROUP BY`

###  Data Analysis in Python
The SQL results were loaded into Pandas for further analysis.

###  Data Visualization
Matplotlib was used to visualize customer spending patterns.

---

##  Key Insights
